Introducción

Las páginas web forman parte de nuestra vida, desde que Berners-Lee publicó la primera página web a finales de 1990 en el CERN, la web no tenía nada que ver con lo que conocemos hoy en día. Las páginas web al principio sólo tenían texto. En cambio, más de 29 años más tarde, se han vuelto muy complejas, aunque la base sigue siendo el hipertexto y la hipermedia junto con los enlaces o hipervínculos.

Dependiendo de las necesidades, el público objetivo y el contenido que vayamos a mostrar, elegiremos un tipo u otro de sitio Web. Esta Web adaptativa en tanto que nos adaptamos a los intereses de los usuario o clientes, deben permitir una flexibilidad, donde si bien el diseño gráfico es importante, no debe nunca supeditarse al contenido.

Existen diferentes clasificaciones que podemos utilizar para definir distintos tipos de sitios Web, según su funcionalidad, a que se dedican, o bien sus características internas. Vamos a ir viendo cada una de esas clasificaciones. Algo básico que debemos saber es la diferencia entre página Web, sitio Web y portal Web. En el cuadro que tenemos en la parte inferior, podemos ver las características de cada uno de ellos.
